# 📚 Amazon Bestseller Books Analysis #Excel

## 📌 Project Overview
Descriptive analysis of 210 Amazon bestseller books (2023–2025) evaluating author dominance, pricing strategies, format trends, and reader sentiment.

## 🎯 Objectives
- Identify top-performing authors & genres
- Analyze format pricing differences
- Study rating vs engagement relationship
- Provide publishing strategy insights

## 📊 Key Findings
- Paperback holds 50.5% market share
- 4.6–4.8 rating band is commercial sweet spot
- Market dominated by few high-volume authors
- Strong correlation between review count and brand recognition

## 🛠 Tools Used
- Excel
- Pivot Tables
- Descriptive Statistics
- Segmentation Analysis

## 💼 Strategic Insights
- Focus on high-volume fiction authors
- Promote niche high-rated books
- Optimize paperback inventory strategy
